# Профиль пользователя

Руководитель проектов в строительстве с фокусом на сметном деле, бюджетном контроле и договорной работе.

## Контекст работы

- Управление бюджетами проектов: калькуляция расценок, план-фактный анализ, контроль отклонений
- Договорная работа: дополнительные соглашения, претензионная переписка, верификация КС-2/КС-3
- Координация субподрядчиков, поставщиков, смежных отделов
- Тендерная документация, коммерческие сметы, исполнительная документация

## Программное обеспечение

- MS Excel — основной инструмент для бюджетов и анализа
- 1С — управленческий учёт, выгрузки данных
- MS Project — графики производства работ
- AutoCAD — чтение проектной документации
- Python 3.11 — для автоматизации (openpyxl, pandas, python-docx)

## Как работает

- Присылает один или несколько файлов + описывает задачу словами
- Нужен готовый файл на выходе (Excel, Word, PDF) — не объяснения и не псевдокод
- Если нужен скрипт — он должен быть готов к запуску и сразу создавать выходной файл

## Предпочтения

- Язык ответов: русский
- Стиль: кратко и по делу
- Не задавать лишних уточняющих вопросов — делать по тому что есть, если не хватает данных — спросить одним вопросом
